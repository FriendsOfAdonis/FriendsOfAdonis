Lucidity is a library for [Lucid ORM](https://lucid.adonisjs.com/) that automatically generate migrations based on your models changes.

```ts tab="Model"
export default class Post extends BaseModel {
  @column.uuid({ isPrimary: true })
  declare id: string

  @column.string()
  declare title: string

  @column.text({ nullable: true })
  declare description: string | null

  @column.updatedAt()
  declare updatedAt: DateTime

  @column.createdAt()
  declare createdAt: DateTime
}
```
