`@foadonis/actions` provides an optional OpenTelemetry instrumentation that automatically creates spans for each action execution. This integrates with the `@adonisjs/otel` package to give you visibility into your actions' performance.

## Setup

First, make sure you have `@adonisjs/otel` installed and configured. See the [AdonisJS OpenTelemetry guide](https://docs.adonisjs.com/guides/digging-deeper/opentelemetry) for setup instructions.

Then add the telemetry hook to automatically span your actions.

```ts title="start/kernel.ts"
import actions from '@foadonis/actions/services/main'
import { telemetryHook } from '@foadonis/actions/otel'

actions.hook('execute', telemetryHook)
```

That's it. Your actions will now be traced automatically, creating spans with the action name and capturing any errors that occur during execution.
