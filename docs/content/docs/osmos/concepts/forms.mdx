Osmos makes it easy to build reactive forms with server-side validation and complex logic.

## Simple Form

You can create simple forms by using the attributes `$model` and `$submit`.

```tsx twoslash
// @noErrors
import { Component, RefAccessor } from '@foadonis/osmos'
import vine from '@vinejs/vine'

const validator = vine.compile(
  vine.object({
    title: vine.string(),
    content: vine.string(),
  })
)

export class CreatePost extends Component {
  title = ''
  content = ''

  render(that: RefAccessor<CreatePost>) {
    return (
      <form $submit={that.create}>
        <input $model={that.title} />
        <input $model={that.content} />
        <button type="submit">Create</button>
      </form>
    )
  }

  create() {
    const { title, content } = await validator.validate({
      title: this.title,
      content: this.content,
    })

    const post = await Post.create({
      title: this.title,
      content: this.content,
    })

    this.redirect(`/posts/${user.id}`)
  }
}
```

In the above example we use `$model` to create a two-way data binding. And `$submit` to call our action when the form is submitted.

## Form Helper

As forms are an important part of all web application we created a helper that manages validation, default values, errors, etc.

```tsx twoslash
// @noErrors
import { Component, RefAccessor } from '@foadonis/osmos'
import vine from '@vinejs/vine'

const validator = vine.compile(
  vine.object({
    title: vine.string(),
    content: vine.string(),
  })
)

export class CreatePost extends Component {
  form = Form(validator, {
    title: '',
    content: '',
  })

  render(that: RefAccessor<CreatePost>) {
    return (
      <form $submit={that.create}>
        <input $model={that.form.title} />
        {this.form.error('title')}
        <input $model={that.form.content} />
        {this.form.error('content')}
        <button type="submit">Create</button>
      </form>
    )
  }

  create() {
    const { title, content } = await this.form.validate()

    const post = await Post.create({
      title: this.title,
      content: this.content,
    })

    this.redirect(`/posts/${user.id}`)
  }
}
```

### Resetting form

A Form can be reset with initial values using `reset()` method.

```tsx
this.form.reset()
```
